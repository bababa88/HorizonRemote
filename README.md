HorizonRemote
=============

UPC Horizon ( Samsung SMT-G7400 ) WIFI-remote for Android.

![Screenshot][2]

This application has been designed to work with Android 4.0 ( sdk level 14 ) and later.

**See [xda][1] for more information.**
